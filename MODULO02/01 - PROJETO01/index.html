JOGO DA ADIVINHAÇÃO

Regras

O jogo funciona da seguinte maneira. O usuario irá escolher um número de 0 até 10. Enquanto isso o computador irá gerar um número aleatório de 0 até 10. O objetivo do jogo é que o usuario acerte o número gerado aleatóriamente pelo computador

Padrão

Essa é a tela padrão que aparece durante o jogo.

Acertou

Caso você acerte o número que o computador gerou aleatoriamente, a seguinte imagem ira aparecer para você

Errou

Caso você erre o número que o computador gerou aleatoriamente, a seguinte imagem ira aparecer para você.

O jogo

JOGUE AGORA

Clique no botão para gerar o número do usuario

Clique no botão para gerar o número do computador

Clique no botão para conferir o resultado
